# 饭团，同事喊你一块儿吃饭
常年跟同一波人吃饭，腻了？
或者，吃饭的，只剩下你一个了？
梦想着有一天能和前排的女神一块儿吃饭？
又或者，只是想听听其它码农在干嘛，增加自己的阅历。
至少，下一次吃饭，不用再纠结去哪了

## 愿景
* 让工作餐变成一件有趣的事情

## 功能
* 约约约：请客吃饭能解决的事情，咋不多说。
* 摇一摇：随机匹配吃饭。
* 小组团：一起腐败去。去哪？投票？Go~Go~Go~
* 兴趣团。谁谁谁包场了？速！度！秒！杀！

## 竞争对手
* 约饭，一个已然沦为约炮的APP。我们采用实名制，从根本上杜绝不良导向，面向企业用户，传递正能量。
* 微信、扣扣，沟通起来太费劲。
